Install as an App

Android (Chrome)

  1. Open any app link above in Chrome
  2. Tap the three-dot menu (top right)
  3. Tap "Install app" or "Add to Home screen"
  4. Done — opens full-screen, works offline

iPhone (Safari only)

  1. Open any app link above in Safari
  2. Tap the Share button (square with arrow, bottom center)
  3. Scroll down and tap "Add to Home Screen"
  4. Done — opens full-screen, works offline

Chrome on iPhone doesn't support app install — must use Safari.
