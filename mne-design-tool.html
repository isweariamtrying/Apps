MNE Design Tool — Help

Getting Started

Drag devices from the sidebar onto the canvas to build your network design. Use Connect mode to wire them together. The summary panel (bottom-left) updates automatically with a BOM and warnings.

Canvas Navigation

PanLeft-click drag on empty canvas, or hold Space + drag
ZoomMouse wheel (zooms toward cursor, 15%–300%)
Fit ViewToolbar button — fits all content in view
Reset ZoomToolbar button — resets to 100% at origin

Modes (Toolbar)

SelectDefault mode. Click nodes/zones/connections to select and see properties. Drag to move. Press Delete to remove the selected item.
ConnectClick a source node, then click a target node to create a connection. Defaults to Ethernet (blue). Multiple connections between the same pair are supported (e.g. for LAGs) — they render as parallel offset lines.

Connection Media Types

Click a connection to select it, then choose the media type in the properties panel. Each type has a distinct color:
Ethernet (blue)Default — copper/Cat6
SMF (yellow)Single-mode fiber
MMF (red)Multi-mode fiber
GameChanger (green)GameChanger links
Connections marked as "Customer Provided" display as dashed lines with a [CP] prefix. You can also add a free-text label (e.g. "Cat6", "OS2") that displays on the connection alongside the media type.

Devices

MX AppliancesFirewalls — shows throughput tier
SwitchesAccess & aggregation switches. Stackable models auto-detect stacks when connected in the same zone.
Access PointsIndoor/outdoor APs. Set quantity and antenna in properties.
CamerasMV cameras with PoE draw shown
SensorsMT sensors (no PoE — battery powered)
Generic DeviceGray node for patch panels, UPS, etc. Connectable and nameable.
PoE DeviceRed node for IP phones, etc. Set name, quantity, and watts per device. Counts toward switch PoE load and port usage.

Zones

Drag MDF, IDF, or Server Room zones onto the canvas to group devices. Drag the zone header to move it, drag the bottom-right corner to resize. Zones show power totals for contained devices in properties.

Stacking

Connect two or more stackable switches of the same family (MS150, C9300L, C9300, C9300X) inside the same zone. A dashed blue outline appears showing combined port pool, PoE budget, and uplink count. Stack cables are auto-added to the BOM.

Switch Properties

NIM ModuleModular switches (C9300, C9300X) can add a NIM for extra uplinks.
Swap ModelChange a switch to a different model without losing connections. All connections, media types, and CP flags are preserved.
AGG ToggleMarks a switch as aggregation — all ports become uplinks in the combined pool.
No Stack (HA)Prevents a stackable switch from auto-stacking with neighbors (e.g. HA pairs in the same zone).
Port BarsGreen bar = port usage, orange bar = PoE load. Turns red above 80%.

Shortcuts

Delete / BackspaceRemove selected node(s), connection(s), or zone
Shift + clickMulti-select nodes or connections (then drag to move all, or change properties as a group)
Ctrl + drag nodeClone device (increments name, keeps settings)
Ctrl + drag zoneClone zone
Ctrl+ZUndo
Ctrl+YRedo

Save / Load / Export

Save / LoadSaves to browser localStorage. Use the sidebar controls.
Export / ImportDownloads/uploads a .json file for sharing designs between users.

Reference Panels

Use the MNE Ref and ENE/ENR/MNR Ref toolbar buttons to open quick-reference overlays with device specs, sizing tables, SFPs, licensing info, and more. Each panel has collapsible sections — click a heading to expand/collapse. Click the background or × to close.

Export Image

Click 📷 Export Image in the toolbar to render the full design as a 2x resolution PNG with a white background. Prompts for a file name, then downloads the image. Useful for pasting into Visio or documentation.

Theme

Use the ☀ Light / 🌙 Dark button in the toolbar to toggle between light and dark mode.
