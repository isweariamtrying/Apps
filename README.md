# My Apps

Personal tools — installable as apps on your phone.

**Browse all apps:** https://isweariamtrying.github.io/Apps/

---

## Install on Android (Chrome)

1. Open the app link in **Chrome**
2. Tap the **three-dot menu** (top right)
3. Tap **"Install app"** or **"Add to Home screen"**
4. Done — opens full-screen, works offline

## Install on iPhone (Safari only)

1. Open the app link in **Safari** (not Chrome — Apple requires Safari for PWA install)
2. Tap the **Share button** (square with arrow, bottom center)
3. Scroll down and tap **"Add to Home Screen"**
4. Done — opens full-screen, works offline

---

## Adding new apps

Drop any .html file into this repo's main branch. The index page auto-detects it and creates a card. The card title comes from the file's <title> tag.
